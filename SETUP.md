## SETUP 

The project will use uv a package manager that make it reproducibility

### import uv 

```pip install uv```

### use of uv

```uv sync```

it will install all the libraries that are in the uv.lock file in the .venv environement just created. you may now use the jupiter noteboock with the kernel of the venv.


### to run tests 

```uv run pytest tests/```

### download MongoDB 
use this link `https://www.mongodb.com/try/download/community` and download `MongoDB Community Server Download`
download the mongosh comand with the link `https://www.mongodb.com/try/download/shell` at `MongoDB Shell Download` and unziped, then add it to the path of the environement path.
